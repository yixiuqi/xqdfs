#### **API协议说明**
* 请求方式：POST
* 请求body字符串格式：JSON
* Content-Type：application/json

<br>

#### **API返回通用字段**
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;API以json格式返回，通用字段如下：

|  字段 | 类型  | 说明  |
| ------------ | ------------ | ------------ |
| result | int  | 返回码，0表示成功，非0表示错误码  |
| info  |  string | API返回信息   |
